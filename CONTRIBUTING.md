# Contributing

Thanks for considering a contribution to Wright. The full contributor
guide lives behind the contributor/user firewall in
[`docs/dev/contributing.md`](docs/dev/contributing.md) and covers:

- Build prerequisites and system libraries
- Running `cargo test` and filtering tests
- `cargo fmt` and `cargo clippy -D warnings` gates enforced by CI
- Conventional Commits style with module scopes
- Pull request workflow and documentation routing

For architectural background before opening a non-trivial change, also
read the [Module Layout](docs/dev/module-layout.md) and the
[ADR index](docs/adr/index.md) — many design choices are documented
there as immutable records.
