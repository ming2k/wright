# ADR-0001: Record Architecture Decisions

## Status

Accepted

## Context

Wright is a complex system with many design choices that affect maintainability. Without a record, future contributors must re-discover the reasoning behind each decision.

## Decision

We will use Architecture Decision Records (ADRs) to document significant design choices. Each ADR captures the context, decision, and consequences.

## Consequences

- New contributors can understand why the system works as it does.
- Proposed changes can be evaluated against recorded rationale.
- The ADR collection becomes a living design document.
